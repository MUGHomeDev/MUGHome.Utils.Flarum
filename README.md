# MUGHome Utils of Flarum API [WIP]

## How to use

- add this to your repositories

```kotlin
    repositories {
        maven("https://mvn.mughome.top/repository/maven-public/")
    }
```

- then add this to your dependency

```kotlin
    dependencies {
         implementation("top.mughome.utils:flarum:0.0.3.2")
    }
```
